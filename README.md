## Descripción del Proyecto

Calculadora web de **10 magnitudes físicas fundamentales**, desarrollada con HTML, CSS y JavaScript puro como parte de la Actividad Evaluativa #3 de Construcción de Algoritmos.

La aplicación permite calcular:

| # | Magnitud | Fórmula |
|---|---|---|
| 1 | Velocidad | v = d / t |
| 2 | Aceleración | a = Δv / Δt |
| 3 | Fuerza | F = m · a |
| 4 | Trabajo | W = F · d · cos(θ) |
| 5 | Energía Cinética | K = ½ · m · v² |
| 6 | Energía Potencial Gravitatoria | U = m · g · h |
| 7 | Densidad | ρ = m / V |
| 8 | Presión | P = F / A |
| 9 | Carga Eléctrica | q = I · t |
| 10 | Ley de Ohm | V = I · R |
